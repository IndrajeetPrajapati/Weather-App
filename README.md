# Weather App

A modern and responsive weather application built using **HTML, CSS, and JavaScript** that provides real-time weather information for any city using a weather API.

## 🌦 Features

* Search Weather by City Name
* Real-Time Temperature Display
* Weather Condition Icons
* Humidity Information
* Wind Speed Details
* Error Message for Invalid City
* Clean and Responsive UI Design
* Fast API Data Fetching

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Weather API

## 📌 Purpose

This project was created to practice API integration, asynchronous JavaScript, fetch requests, DOM manipulation, and responsive frontend development.

## 🚀 How It Works

1. Enter a city name.
2. Fetches live weather data from API.
3. Displays temperature, humidity, wind speed, and condition.
